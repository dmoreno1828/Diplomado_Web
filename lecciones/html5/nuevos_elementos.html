Nuevos Elementos

El doctype, en primer lugar tenemos la declaración del tipo de documento, en el que se trata de una forma sencialla de decirle al navegador, o a cualquier otro analizador, qué tipo de documento esta visualizando. Con esta versión solo tenemos que declararla: 

<!DOCTYPE html>
.

El elemento html, en cualquier documento esta el elemento html, que no ha cambiado significativamente con HTML5, se usa el atributo lang para definir el tipo de lenguaje del documento. En html5 no es necesario incluir el atributo lang para validar nuestro documento. 

<html lagn="es">

El elemento head, dentro de esta etiqueta se incorpora los elementos meta que permite indicar ciertas caracteristica de nuestro documento, como por ejemplo la codificación, autores entre otras. 

<meta charset="UTF-8">
<meta name="description" content="Ejemplo de HTML5" >
<meta name="autho" content="Duglas Moreno" >
Los demas elementos se declaran de la misma manera. 

<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8">
<meta name="description" content="Ejemplo de HTML5" >
<meta name="autho" content="Duglas Moreno" >
<title> Diplomado </title>
</head>
<body>
</body>
</html>
