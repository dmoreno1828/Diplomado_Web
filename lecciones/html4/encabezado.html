Colocando Título a nuestra página

Todo dentro de nuestro archivo HTML irá en medio de las etiquetas de inicio <html> y de cierre </html>.

El archivo siempre consiste de dos partes: el encabezado y el cuerpo. Vamos a empezar con el encabezado.

El encabezado tiene información sobre el archivo HTML, como por ejemplo su título. El título es lo que vemos en la barra de títulos del buscador o pestaña de la página. Por ejemplo, el título de esta página es "Diplomado en Tecnología Web".

Instrucciones

Agreguemos una etiqueta y un título a nuestra página web. Si te trabás en alguna parte hacé clic más abajo en “¿Te trabaste? Te damos un consejo” para ver un ejemplo.

  1. Agregá una etiqueta de inicio <head> y una de cierre </head>.
  2. En medio de las etiquetas <head> de inicio y de cierre, agregá una etiqueta de título <title> de inicio y una etiqueta </title> de cierre.
  3. Entre las etiquetas <title>, escribí un título para tu página web. Por ejemplo, “Mi página web”.
