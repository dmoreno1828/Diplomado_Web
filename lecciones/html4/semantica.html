Semantica del HTML 4

Para aprender más HTML, debemos aprender cómo escribir el HTML. Ya viste que usamos mucho los símbolos <>.

  1. La semantica del HTML comienza con la etiqueta <html> donde se denotara el inicio del codigo del html.
  2. Se usa una etiqueta <head> que indica el inico de la cabecera, donde se puede incluir, otras etiquetas que permiten por ejemplo colocar autor, descripción, codificación de la pagina entre otras caracteristica que veremos mas adelante.
  3. Se usa una etiqueta <body> que indica el inico del cuerpo, donde se puede incluiyen las diferentes etiquetas que permitiran crear nuestra página web.

Estas etiquetas nos permiten construir la sintaxis de un documento en html, por ejemplo:

<html>
<head>
</head>
<body>
<!-- Cuerpo de nuestro documento -->
</body>
</html>

En este ejemplo se muestra la semantica básica de nuestro documento en html.

La práctica hace al maestro. Repitamos:

Instrucciones
  1. Colocá la etiqueta <!DOCTYPE HTML>
  2. Coloca las etiquetas de semantica del Html
  3. Escribí lo que quieras en medio de las etiquetas <body>.
