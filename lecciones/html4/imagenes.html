Imágenes (<img />)

Para insertar la imágen dentro de nuestro documento hay que usar la etiqueta img y el atributo src que se toma del término source. Este atributo lleva como valor la dirección, relativa o absoluta, de la ubicación de la imagen. Por Ejemplo: 

<img src="http://www.ccsl.org.au/~ccslorg/images/YouTubeSquareLogo.png">

Dicha etiqueta tienes varios atributos que permite su manipulación como lo son:

  • alt dicho atributo permite colocar un texto alternativo, que se visualizara si el enlace esta roto.
  • title permite colocar un titulo a la imagen
  • width y height son valores que permite cambiar lo ancho y lo largo

Cabe resaltar que podemos comenzar a unir etiquetas, como por ejemplo, si quiero que una imágen se ha un enlace se hace de la siguiente manera: 

<a href="https://www.youtube.com/">
<img src="http://www.ccsl.org.au/~ccslorg/images/YouTubeSquareLogo.png">
</a>
